Detailed Description

ModelGroup contains a left/right pair of HandModelBase's

Parameters
modelListThe HandModelBases available for use by HandRepresentations
modelsCheckedOutThe HandModelBases currently in use by active HandRepresentations
IsEnableddetermines whether the ModelGroup is active at app Start(), though ModelGroup's are controlled with the EnableGroup() & DisableGroup methods.
CanDuplicateAllows a HandModelBases in the ModelGroup to be cloned at runtime if a suitable HandModelBase isn't available.
